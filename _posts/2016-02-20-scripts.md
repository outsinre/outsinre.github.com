2. bulk rename

    ```bash
    a=1
    for i in *.jpg; do
      new=$(printf "%04d.jpg" "$a") #04 pad to length of 4
      mv -- "$i" "$new"
      let a=a+1
    done
    ```
3. number range

    ```bash
    # -w means equal width
    $ seq -w 5 20
    ```
