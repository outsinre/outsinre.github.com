This is a GDB tutorial to debug program.

# gef #

[gef](https://github.com/hugsy/gef) is a Python extension added to GDB.

Download gef.

```bash
~ $ git clone https://github.com/hugsy/gef
```

Load gef on the fly. Alternatively, add `source ~/workspace/gef/gef.py` to `~/.gdbinit`.
