###1 ctex xecjk fontspec

ctex 依赖 xeCJK 依赖 fontspec，一个比一个底层。

ctex 管中文格式；
xeCJK 管中西文字体自动切换、中文标点压缩；
fontspec 管任意字体选择。

###2 Code by normal class

{% highlight TeX linenos %}

\documentclass{article}
\usepackage{ctex}
\begin{document}
这是我的第一篇中文排版文件。
\end{document}

{% endhighlight %}
