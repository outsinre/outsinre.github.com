# Outline

1. VMess;
2. Websocket;
3. TLS;
4. Nginx proxies V2ray by URL (*/wss*).

   To-dos: by HTTP *header* is better.

>Comments (`//foo` or `/* bar */`) are accepted in configuration.

# Nginx proxy

```
  ssl_certificate      /etc/letsencrypt/live/www.example.com/fullchain.pem;
  ssl_certificate_key  /etc/letsencrypt/live/www.example.com/privkey.pem;   

  location /wss {
	proxy_redirect off;
        proxy_pass http://127.0.0.1:1234;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $http_host;
  }
```

1. Use *fullchain.pem* [instead of](https://github.com/v2ray/v2ray-core/issues/509#issuecomment-319321002) *cert.pem*.
2. Proxy port should be the same as server side listening port.
